# Install tyronZIL-js

1. ```git clone https://github.com/julio-cabdu/tyronZIL-js```

2. ```cd tyronZIL-js```

3. ```git status```     // if you wanna start contributing, then create your topic branch: ```git checkout -b tyron```

4. ```npm install```

5. ```npm run build```

6. ```npm install -g .```   // to get the CLI ready
